# Al-Trading-App---BTC-Gold-Trend-Prediction
This app uses machine learning to predict short-term trends for Bitcoin (BTC-USD) and Gold (GC=F). It analyzes technical indicators such as SMA, EMA, and RSI, retrieves real-time market data, and provides a simple up or down trend prediction through a Streamlit web interface.
# AI Trading App – BTC & Gold

## Overview
AI Trading App is a machine learning-based web application that predicts short-term market trends for Bitcoin (BTC-USD) and Gold (GC=F). The app uses technical indicators (SMA, EMA, RSI) to analyze market data and provide simple up/down trend predictions.

## Features
- Real-time BTC and Gold trend prediction
- Calculation of technical indicators (SMA, EMA, RSI)
- Machine learning model (Random Forest)
- Simple Streamlit web interface

## Tech Stack
- Python
- Streamlit
- Pandas & NumPy
- Scikit-learn
- yFinance
- Joblib
